# Bioinformatics & Data Science Pipelines

A collection of customized pipelines for deep learning, genomics, neuroimaging, and automated reporting tasks.

## Pipeline Overview

### Deep Learning
- **Location**: `DL/`
- **Description**: End-to-end pipelines for building and training deep learning models
- **Features**: Model architectures, training loops, hyperparameter optimization

### GWAS Analysis
- **Location**: `GWAS/`
- **Description**: Genome-wide association study pipelines
- **Features**: Quality control, association testing, result visualization

### Neuroimaging
- **Location**: `MRI/`
- **Description**: Structural and functional MRI analysis pipelines
- **Features**: Preprocessing, cortical thickness, functional connectivity

### Polygenic Scores
- **Location**: `PGS/`
- **Description**: Polygenic score calculation and validation
- **Features**: Score computation, normalization, association testing

### Audio Transcription
- **Location**: `audio-tx/src/`
- **Description**: Automated audio transcription using OpenAI Whisper
- **Features**: Batch processing, multiple formats, output customization

### FreeSurfer Processing
- **Location**: `freesurfer/`
- **Description**: Automated FreeSurfer pipeline with parameter variations
- **Features**: Multiple recon-all configurations, batch processing, quality control

### Automated Literature Updates
- **Location**: `labtv/`
- **Description**: Automated Google Slides creation for recent publications
- **Features**: Weekly updates, keyword-based filtering, auto-generated presentations

### Low-Pass WGS Variant Calling
- **Location**: `low-pass-WGS-variant-calling/`
- **Description**: Variant calling from low-coverage whole genome sequencing
- **Features**: Quality control, variant detection, genotype refinement

### Transcriptome Imputation
- **Location**: `tx-imputation/`
- **Description**: Tissue-specific transcriptome imputation using genotype and eQTL data
- **Features**: Multiple tissue models, prediction accuracy assessment
